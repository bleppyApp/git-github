Git Basic Workflow

Command Line Basic Workflow Git Commands Highlights Merges
CREATE A FOLDER, IF NEEDED:

mkdir name_of_folder

GO TO FOLDER, IF NEEDED:

cd name_of_folder

MAKE SURE GIT WASN'T INITIATED:

git status

INITIATE GIT:

git init

CREATE FILES, IF DOING IT FROM THE COMMAND LINE:

touch file.js

DO THE WORK
CHECK CHANGES:

git status

STAGE FILES:
  1. Single or multiple files:

    git add file1.js file2.js

  2. All files at once

    git add .

CHECK IF CHANGES WERE STAGED:

git status

COMMIT CHANGES:

git commit -m "message about the commit"
