New Repo Workflow

  1. Access Github

    2. On github, click on add repository and create a new repository.

  2. Set up the remote on VS code

    • Make sure you already initiated git on the folder: git status to check and git init to initiate it.
    • Make sure you already don't have a remote: git remote -v
    • Add a new remote: git remote add origin URL
    • Push the new project: git push --set-upstream origin master
