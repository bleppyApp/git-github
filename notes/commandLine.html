COMMAND LINE

Command Line Basic Workflow Git Commands Highlights Merges
  • ls

    list of content at the current location (folder)

  • ls nameOfFolder

    lists the content inside that folder, but only if the folder is inside the current location. It won't work to a folder that is inside another folder. To do that, you'll need a path: ls nameOfFolder/nameOfInsiderFolder

  • ls -a

    lists hidden files

  • start .

    opens the file explorer

  • start nameOfFolder

    opens folder graphically

  • clear

    cleans the terminal

  • pwd

    prints the path from your current location

  • cd

    moves to a different folder: cd nameOfFolder

  • cd ~

    goes to the root directory

  • cd ..

    Goes back to the previous folder

  • touch nameOfNewFile.extension

    creates a new file in the current location (e.g.: touch file.pdf). To create in a different location, you can cd to the location or enter the path to that location. (e.g.: touch Desktop/folder/newFileName.txt)

  • mkdir nameOfFolder

    creates a new folder (directory) in the current location. As always to create a directory in another location, you can cd to that location or enter the path to that location.

  • rm nameOfFile

    deletes PERMANENTLY the file

  • rm -rf nameOfFolder

    deletes PERMANENTLY a folder

  • cd d:

    moves to local disk d

  • Rename a folder

    mv oldName/ newName

  • ctrl U

    deletes whatever was typed in the CLI

  • ctrl a

    goes to the beginning of the typed line

  • ctrl e

    goes to the end of the typed line

  • alt click

    makes the mouse work inside the CLI
