Merge Workflow

Command Line Basic Workflow Git Commands Highlights Merges

  • Fast-forward merge:

    When there is only one new branch and all the changes are made there. Then, all you need is to merge the new branch to the master.

    1. Switch to the branch you want to be merged, the destination. In this case, to master. (git switch master)
    2. Merge the new branch. (git merge newBranchName)

  • Resolving Conflicts

    Whenever you encounter merge conflicts, follow these steps:

    1. Open up the conflicting file(s)
    2. Solve the conflicts however is best
    3. Remove the conflict marks that were added by GIT
    4. Stage and commit changes
