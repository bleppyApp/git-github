New Project Workflow

* Front-end

  1. Open VS code
  2. Cd to location where the project will be created
  3. Create a new app and cd to it
  4. Make sure git was not initiated: git status
    5. If using npx create-expo-app dguys-mobile it'll automatically initiate git
  5. Initiate git: git init
  6. Connect it to Github, following the New Repo Workflow
  7. Add redux toolkit
  8. Add navigation
  9. Add bootstrap or react paper

* Back-end

  1. npm install -g firebase-tools
  2. npm install firebase-functions
  3. firebase init functions
  4. Check if project has environment variables set: functions:config:get
  5. Set environment variables, if apply: firebase functions:config:set object_name.key=value
