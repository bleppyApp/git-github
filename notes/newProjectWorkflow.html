New Project Workflow

  1. Open VS code
  2. Cd to location where the project will be created
  3. Create a new app and cd to it
  4. Make sure git was not initiated: git status
    5. If using npx create-expo-app dguys-mobile it'll automatically initiate git
  5. Initiate git: git init
  6. Connect it to Github, following the New Repo Workflow
